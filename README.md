📦 API de Cadastro de Clientes e Produtos

API REST desenvolvida para realizar o cadastro, consulta, atualização e exclusão de clientes e produtos. O objetivo é fornecer um backend simples e escalável para sistemas de gestão, e-commerce ou aplicações administrativas.

🚀 Tecnologias Utilizadas

Node.js – Ambiente de execução JavaScript

Express.js – Framework para criação de APIs REST

Para rodar o projeto, execute o seguinte comando: node index.js
